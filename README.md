## Projects

### 1. [Job Board](https://github.com/Aman-Yadav-13/job-board)
A platform where job seekers and employers can connect. Job seekers can view job posts, and employers can post job opportunities. The project includes:
- Frontend built with React
- Backend using Node.js and Express
- Integration with MongoDB
- Deployed on Render

### 2. [Todo List Application](https://github.com/Aman-Yadav-13/todo-list)
A React/Next.js application for managing tasks with features like task creation, updates, completion, and search. Utilizes server-side rendering (SSR) and URL parameters.

### 3. [Speech Recognition Color Changer](https://github.com/Aman-Yadav-13/speech-recognition-color-changer)
A web application that listens to speech, recognizes colors, and updates the color of an image on the website accordingly.
